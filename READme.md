# Network Analysis Basics

This project demonstrates my understanding of fundamental networking concepts through practical command-line analysis.

## Topics Covered
- ARP (Address Resolution Protocol)
- Traceroute (Routing path analysis)
- Netstat (Active connections and ports)

## Tools Used
- arp
- tracert / traceroute
- netstat

## Objectives
- Understand how devices communicate in a LAN
- Analyze packet flow across networks
- Identify active connections and services

## Files
- arp-analysis.md → ARP table behavior and observations
- traceroute-analysis.md → Network path analysis
- netstat-analysis.md → Port and connection analysis

## Key Learnings
- ARP resolves IP addresses to MAC addresses within a LAN
- Data travels through multiple routers (hops) before reaching destination
- Systems maintain multiple active connections using ephemeral ports
